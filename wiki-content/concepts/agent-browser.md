# Agent Browser

Agent Browser 是更适合登录态抓取、页面操作和 state 保存的浏览器自动化工具。

## 关键点
- 适合页面打开、点击、输入、截图、抓取文本。
- 支持保存 / 恢复登录态。
- 对需要登录后抓全文的网页更有帮助。
