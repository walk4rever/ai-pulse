## 📦 本周焦点

### Claude Code 源码泄露：500k+ 行代码引发的架构革命

3月31日，Claude Code 源代码通过 npm registry 的 `.map` 文件意外暴露，约 **512k 行 TypeScript** 代码被公开。这是首次大规模审视生产级 AI Coding Agent 的内部实现。

**核心架构发现**：

| 组件 | 详情 |
|------|------|
| **三层记忆系统** | 1) MEMORY.md 仅作索引 2) 主题文件按需加载 3) 完整会话记录可搜索 |
| **子代理 KV Cache** | Fork-join 模型，并行化几乎零成本 |
| **5级权限系统** | 精细化的工具调用控制 |
| **Plan Mode 两种模式** | TodoWrite / TaskOutput 等工具的差异化设计 |
| **Resilience/Retry** | 内置错误重试和恢复机制 |

> "The bar for what 'production-grade' means just got documented in detail."

**市场反应**：
- 某 fork 迅速达到 **32.6k stars、44.3k forks**
- Anthropic 发起 **DMCA 下架**但去中心化镜像已遍布
- 社区评价从"尴尬泄露"转为"架构盛宴"

## 🧑‍💻 Coding Agent

### Hermes Agent：开源 Agent 框架新王者

本周最耀眼的开源新星。多个知名开发者公开从 OpenClaw 迁移到 Hermes：

**核心优势对比**：

| 维度 | OpenClaw | Hermes Agent |
|------|----------|--------------|
| 技能构建 | Human-authored skills | Self-forming skills |
| 记忆系统 | Markdown memory | Persistent/searchable memory stacks |
| 架构模式 | Gateway control plane | Self-improving loop |
| 经济模型 | $20/$200 订阅 | 不匹配 24/7 工作负载 |

> "The edge is not just the model, but the harness + learning loop." — @supernovajunn

**生态爆发**：
- Manim skill：生成技术动画和解释视频
- Hermes HUD：实时进程映射到 tmux pane
- Slash-command 技能加载支持 Discord/Telegram

### 开源数据新基建：pi-share-hf

@badlogicgames 发布 **pi-share-hf**，将 Coding Agent 会话发布为 Hugging Face 数据集（内置 PII 防御）。

@ClementDelangue（HF CEO）明确表示：
> 这是开源前沿 Agent 缺失的关键拼图——社区已生成大量 traces，数据应该众包共享。

**意义**：Self-improving models 将可直接从生产 traces 学习，而非依赖干净沙盒。
