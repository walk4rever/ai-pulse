## 📦 本周焦点

### OpenAI 完成 $110B 融资：史上最大规模 Startup 融资

OpenAI 宣布完成 **$110B** 融资轮，由 Amazon、NVIDIA 和 SoftBank 联合领投，投后估值达 **$840B**。这是人类历史上最大规模的创业公司融资。

**核心条款**：
- Amazon 投资 **$50B**（首轮 $15B + 后续 $35B），成为 OpenAI 独家第三方云提供商，获得 2 GW Trainium 算力（$100B/8年）
- NVIDIA 投资 **$30B**，包含 3 GW 推理算力 + 2 GW Vera Rubin 训练算力
- SoftBank 投资 **$30B**

> "The round would nearly triple total capital raised to date" — Epoch AI

与此同时，The Information 披露 OpenAI 预计 **2028 年前烧掉 $157B 现金**，本轮融资大致匹配这一需求。

### Anthropic $30B ARR + Claude Mythos：首个"因危险而不发布"的模型

Anthropic 宣布年化收入突破 **$30B**（从 2025 年底的 $9B 增长），同时发布 **Claude Mythos**（来自希腊语"叙事"）——这是自 GPT-2 以来首个因**过于危险**而不公开发布的模型。

- 仅授权给 **40 个合作伙伴**（Project GlassWing）
- 244 页 System Card 详述其"惊人能力"
- 与 Google 和 Broadcom 签署 **多 GW 下一代 TPU 算力**协议（2027 年起）

### Claude Code 源码泄露：50万行代码揭示 Agent 架构

Claude Code 源码因 source map 泄露，暴露超过 **500K 行 TypeScript** 代码，引发社区疯狂逆向工程。核心发现：

- **Memory 三层设计**：MEMORY.md 索引 + 按需加载 topic 文件 + 可搜索会话 transcript
- **Subagent fork-join 模型**：利用 KV cache 实现并行，理论" parallelism is basically free"
- **5 级权限系统**
- **2 种 Plan 模式**

> "what does this reveal about state-of-the-art agent harness design?"

## 🌍 开源模型

### Gemma 4：iPhone 上跑 40 tok/s

Google 发布的 Gemma 4 成为小模型最大突破：
- **2M 下载量**
- 在 **iPhone 17 Pro** 上用 MLX 达到 **~40 tok/s**（@adrgrondin）
- 老旧硬件（2013 Dell E6440）也能跑 E2B 版本，达到 **8 tokens/s**
- 31B 模型需 **40GB VRAM**（MoE + KV Cache）
- Apache 2.0 许可证

| 模型 | 参数量 | 活跃参数 | 目标场景 |
|------|--------|----------|----------|
| Gemma 4 31B | 31B | 31B | 数据中心 |
| Gemma 4 26B A4B | 26B | ~4B | 高吞吐服务器 |
| Gemma 4 E4B | - | ~4B | 边缘/IoT |
| Gemma 4 E2B | - | ~2B | 移动端 |

### GLM-5.1 冲击 Code Arena 前三

Z.ai 的 **GLM-5.1** 登上 Code Arena **#3**，超越 Gemini 3.1 和 GPT-5.4，与 Claude Sonnet 4.6 持平。Z.ai 同时发布 **GLM-5V-Turbo**，原生支持图像/视频/文档布局/设计稿处理。

### PrismML 1-bit Bonsai：1.15GB 的端侧模型

PrismML 发布 **Bonsai 8B**（1-bit 权重模型）：
- 仅需 **1.15 GB** 内存
- **14x 更小**，**8x 更快**，**5x 更节能**
- Apache 2.0 开源

### TII Falcon Perception + OCR

TII 发布 Falcon Perception（开词汇指代表达分割模型）和 **0.3B OCR 模型**，竞争力比肩 3-10x 大模型。核心设计：**early-fusion transformer** 从第一层就混合图像和文本。

## 🔬 架构研究

### Agent Harness 设计的崛起

本周核心洞察：**框架设计 > 模型本身**。顶级模型的致命弱点是使用联想而非逻辑、模式匹配而非实验。

**NLAHs (Natural-Language Agent Harnesses)**：arXiv 新论文将 agent harness 的高层控制逻辑外部化为可移植的可执行 artifact。这是 Agent 工程走向科学化的里程碑。

### 模型路由成为产品痛点

> "tools like Claude Code and Codex remain too provider-bound"

从业者越来越希望在单一工作流内共享上下文 + 自动路由 + 跨模型协作，而非手动切换终端。

## 🔧 基础设施

### NVIDIA 4-bit 量化：99.4% 精度保持

NVIDIA 论文披露 **Quantization-Aware Distillation (QAD)** 方法，实现 **16-bit 到 4-bit** 压缩，同时保持 **99.4% 准确率**，基本实现无损。

### vLLM ROCm AMD GPU 加速

vLLM 宣布 **7 个 ROCm attention 后端**，通过 KV-cache 布局优化、批处理技巧和模型特定内核，在 AMD GPU 上实现 **最高 4.4x 解码吞吐提升**（环境变量 `VLLM_ROCM_USE_AITER=1`）。

### LMCache：KV Cache 复用

LMCache 被highlight为 KV cache 管理层，可复用重复片段（不仅是前缀），在部分 RAG 场景实现 **4-10x  reduction**，已集成到 NVIDIA Dynamo。

### H100 价格上涨

H100 需求持续强劲，**价格持续上涨**。
