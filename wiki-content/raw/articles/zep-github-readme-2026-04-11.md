# Zep GitHub README Extract

Source: https://github.com/getzep/zep
Retrieved: 2026-04-11

## Key excerpts
- Zep is an **end-to-end context engineering platform**.
- It delivers the right information at the right time with **sub-200ms latency**.
- Zep assembles relationship-aware context from multiple sources such as chat history, business data, documents, and app events.
- The workflow is described as **add context → graph RAG → retrieve & assemble**.
- Zep Cloud is positioned as a managed service with enterprise-grade scalability and compliance.
- SDKs are available for Python, TypeScript/JavaScript, and Go.
