# Letta GitHub README Extract

Source: https://github.com/letta-ai/letta
Retrieved: 2026-04-11

## Key excerpts
- Letta is presented as a platform for building agents with **advanced memory** that can learn and self-improve over time.
- The project was formerly known as **MemGPT**.
- Letta Code runs agents locally in the terminal, while Letta API is used to build agents into applications.
- The README emphasizes **stateful agents** and memory blocks.
- The SDKs mentioned include TypeScript/Node.js and Python via `letta-client`.
