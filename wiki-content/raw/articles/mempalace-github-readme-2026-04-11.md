# MemPalace GitHub README Extract

Source: https://github.com/milla-jovovich/mempalace
Retrieved: 2026-04-11

## Key excerpts
- "The highest-scoring AI memory system ever benchmarked. And it's free."
- MemPalace stores conversations as **raw verbatim storage** in ChromaDB without summarization or extraction.
- The architecture organizes memory into **wings, halls, and rooms** (the "Palace" model).
- **AAAK** is an experimental lossy abbreviation dialect for packing repeated entities into fewer tokens.
- The repository description emphasizes **local, open, adaptable** operation without external APIs or services.
- `pyproject.toml` lists primary runtime dependencies: `chromadb>=0.5.0,<0.7` and `pyyaml>=6.0,<7`.
- The CLI entry point is `mempalace`.
