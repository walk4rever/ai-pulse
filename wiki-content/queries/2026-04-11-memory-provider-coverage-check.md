# 2026-04-11 Memory provider coverage check

## 结论
当前 wiki 里的 memory provider 对比页已经从“代表性子集”扩展到**更接近主流头部项目的集合**，但仍然**不算严格意义上的完整市场地图**。

## 已覆盖的 11 个 provider
- Honcho
- OpenViking
- Mem0
- Zep
- Letta
- Hindsight
- Holographic
- RetainDB
- ByteRover
- Supermemory
- MemPalace

## 这次补进来的头部项目
- **Zep**：end-to-end context engineering platform，强调 relationship-aware context assembly 和 temporal knowledge graph。
- **Letta**：formerly MemGPT，强调 stateful agents 和 advanced memory。

## 目前的覆盖特征
这份表现在已经包含了多条主流路线：
- 云端 SaaS
- 本地 / 自托管
- SDK / API 型
- agent memory / context engineering platform 型
- raw verbatim storage 型

## 仍然可能值得补充的方向
如果后续继续完善“主流全景图”，还可以继续找：
- 更偏企业平台化的 memory/context 产品
- 更偏开源 SDK 的 memory layer
- 更偏知识图谱 / long-term context 的方案
- 已进入主流讨论但尚未入表的项目
